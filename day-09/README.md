# Django Capstone

Today is a project build day. Build a robust project of your choice.

## Deliverables

- Before you start, make a plan! User stories, model relationships, etc.
- Build the project in Django
- Two models and additionally a User model
- At least one model should associate with the User model
- At least one model should have full CRUD
- A User should be able to sign up / log in / log out

## Recommendations

- Add styling, an LLM like Claude can quickly generate styling for those who want to focus on other tasks
- Allow for image uploads for a model
- Create API routes for data so it can be fetched from external sources
